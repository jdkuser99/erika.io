PROGRAMA DE DESARROLLO

HTML - CSS - JAVASCRIPT
Es un lenguaje de marcado que se utiliza para estructurar y organizar el contenido de una página web, utilizando etiquetas para definir elementos como encabezados, párrafos, enlaces, entre otros. Es un lenguaje de estilo que se utiliza para dar estilo y presentación a las páginas web. Permite controlar el diseño, la apariencia y la disposición de los elementos HTML mediante reglas que especifican propiedades como colores, fuentes y márgenes. Es un lenguaje de programación que se utiliza para hacer que las páginas web sean interactivas y dinámicas. Permite manipular el contenido, responder a eventos del usuario y modificar el comportamiento de la página en tiempo real.
